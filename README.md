# Vuejs ASCII Art Generator

A simple vuejs app that converts text to ascii art using figlet

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
